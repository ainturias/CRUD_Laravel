# CRUD Básico en Laravel 🐘

En este repositorio hemos realizado un CRUD (Create, Read, Update, Delete) completo utilizando el framework Laravel de PHP.

## 🚀 Tecnologías utilizadas
- PHP
- Laravel Framework
- Base de datos relacional (MySQL/PostgreSQL)

## 📌 Objetivo
Demostrar las capacidades de Eloquent ORM, el manejo de rutas y controladores, y la creación de vistas con Blade en Laravel.
